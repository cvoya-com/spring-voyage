# Archived documentation

Superseded planning records, kept for historical context. **These do not describe the current system** — for that, see [docs/architecture/](../architecture/README.md) and the [guides](../guide/README.md). Forward-looking work and live progress are tracked on the GitHub [milestones](https://github.com/cvoya-com/spring-voyage/milestones) and in issues.

- [`roadmap/`](roadmap/README.md) — the phase-based roadmap from the superseded release frame that preceded v0.1 (scrapped 2026-04-25; the project refounded as v0.1). The phase snapshots and the forward-looking "future work" narrative are retained for context only.
- [`plan/v0.1/`](plan/v0.1/README.md) — the v0.1 plan-of-record (areas A–J, decision log, cross-cutting umbrellas). v0.1 shipped as the `1.0.0-alpha` family; the active plan frame is now [`docs/plan/v0.2/`](../plan/v0.2/README.md).
