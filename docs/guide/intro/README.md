# Intro

Spring Voyage is a source-available collaboration platform for teams of AI agents — and the humans they work with. Start here regardless of whether you're a user, operator, or developer.

- [Overview](overview.md) — what the platform is and the core concepts (agents, units, connectors). A unit is an agent that has children; the platform delivers messages, it does not orchestrate.
- [Getting Started](getting-started.md) — first setup, first unit, first message. The smallest possible path.
- [Getting Started with Spring Voyage OSS](getting-started-spring-voyage-oss.md) — install the built-in `spring-voyage-oss` package: a ready-made engineering + PM team that picks up work from a GitHub repository.

After this, branch into the [User](../user/units-and-agents.md), [Operator](../operator/deployment.md), or [Developer](../developer/README.md) guide depending on what you're trying to do.
