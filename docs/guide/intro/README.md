# Intro

Spring Voyage is an open-source collaboration platform for teams of AI agents — and the humans they work with. Start here regardless of whether you're a user, operator, or developer.

- [Overview](overview.md) — what the platform is and the core concepts (agents, units, connectors). A unit is an agent that has children; the platform delivers messages, it does not orchestrate.
- [Getting Started](getting-started.md) — first setup, first unit, first agent.

After this, branch into the [User](../user/units-and-agents.md), [Operator](../operator/deployment.md), or [Developer](../developer/README.md) guide depending on what you're trying to do.
