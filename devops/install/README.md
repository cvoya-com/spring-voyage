# devops/install/

Source-free operator install path. Populated by [#2180](https://github.com/cvoya-com/spring-voyage/issues/2180).

For today, operators bring Spring Voyage up by either cloning this repository and
running [`devops/build/build.sh`](../build/README.md) followed by
[`devops/deploy/deploy.sh up`](../deploy/README.md), or by downloading the
release bundle directly — see [`docs/guide/operator/deployment.md`](../../docs/guide/operator/deployment.md).
